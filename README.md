O **Bash Search Pro** é um ecossistema de busca de comandos ultra-rápido integrado à **Sentinela IA** (Google Gemini 1.5 Flash), desenvolvido para ser o companheiro definitivo de desenvolvedores que utilizam **Termux** e sistemas **Linux**.

---

## 🚀 Demonstração em Tempo Real
Acesse a aplicação oficial:
**[https://bash-eee21.web.app](https://bash-eee21.web.app)**

## ✨ Funcionalidades Pro

* 🤖 **Sentinela IA:** Inteligência artificial integrada para explicar e gerar scripts complexos.
* 📱 **Progressive Web App (PWA):** Instalável no Android com ícone personalizado.
* ✈️ **Offline First:** Acesso à base de dados local mesmo sem conexão.
* 📋 **Click-to-Copy Inteligente:** Toque no comando para copiá-lo instantaneamente.
* 🛡️ **SRE Resiliente:** Backend com tolerância a falhas e segurança Firestore blindada.

## 🛠️ Stack Tecnológica

- **Frontend:** HTML5, CSS3 (Tailwind), JS Vanilla, Service Workers (PWA).
- **Backend:** Node.js, Express, Gemini AI API.
- **Cloud:** Firebase Hosting (Frontend) & Render (API Backend).

# 🤖 Sentinela-Bash IA (v2.3)

> **Unidade de Processamento CLI do projeto Minha IA Memória.**

Sistema híbrido para usuários de **Termux/Linux** que combina busca local de comandos e inteligência artificial avançada.

## 🚀 Funcionalidades
- **IA Sentinela:** Integração com Google Gemini 1.5 Flash.
- **Busca Rápida:** Filtro instantâneo de comandos via arquivos locais.
- **SRE Resiliente:** Backend tolerante a falhas com modo degradado.
- **PWA:** Instalável no Android.

## 🛠️ Stack Tecnológica
- **Backend:** Node.js, Express.
- **Frontend:** Tailwind CSS, Marked.js.
- **Cloud:** Firebase Hosting & Render API.
